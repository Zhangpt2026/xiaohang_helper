# 📚 小航 · 校园信息查询 AI 助手

郑州航空工业管理学院人工智能专业认知实习项目——基于 Prompt 工程 + AI API 调用的校园信息查询助手。

## ✨ 功能特性

- **💬 智能问答**：基于学校资料，AI 精准回答校园相关问题
- **🎭 身份分流**：支持新生、在校生、教师三种身份，不同身份不同回答风格
- **💡 推荐问题**：每类身份提供 4 个高频问题，点击快速提问
- **📞 电话黄页**：静态兜底页，API 挂了也能查电话
- **🌙 深色模式**：支持亮色/暗色主题切换
- **📜 问答历史**：记录对话历史，支持导出

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Streamlit 1.35.0+
- 硅基流动 API Key

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd xiaohang_helper

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env，填入你的 API Key

# 启动应用
streamlit run src/app.py
```

### 配置说明

在 `.env` 文件中配置以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `API_KEY` | 硅基流动 API 密钥 | 必须填写 |
| `API_URL` | API 地址 | `https://api.siliconflow.cn/v1/chat/completions` |
| `MODEL_NAME` | 模型名称 | `Qwen/Qwen2.5-7B-Instruct` |

## 🎯 使用说明

1. **选择身份**：在顶部选择你的身份（新生/在校生/教师）
2. **提问**：
   - 点击推荐问题快速提问
   - 或在输入框中输入自定义问题
3. **查看回答**：AI 会基于学校资料给出回答，并标注来源
4. **查看电话黄页**：切换到「电话黄页」标签页查看常用电话

## 📝 Prompt 工程

### 身份分流策略

- **新生**：像热心学长一样详细解答，口语化，多鼓励
- **在校生**：简洁回答，重点：地点、电话、材料、时间
- **教师**：专业礼貌，重点：政策依据、办事窗口、联系人

### 防幻觉硬规则

1. 只能根据资料回答，资料里没有的明说并给出咨询电话
2. 严禁编造电话号码、地址、办公时间、学费金额、人名
3. 涉及金钱/转账，无条件提示防诈骗
4. 涉及心理危机，立即提供心理援助热线
5. 不接入学校系统，拒绝查询个人信息
6. 回答末尾标注来源文件名
