# 办事流程指南

## 一、学籍证明

### 1.1 在校学籍证明
- **办理地点**：教务处学籍科（行政楼203室）
- **所需材料**：学生证、身份证
- **办理时间**：周一至周五 8:30-11:30, 14:30-17:30
- **办结时间**：当场办理

### 1.2 成绩证明
- **办理地点**：教务处成绩科（行政楼205室）
- **所需材料**：学生证、身份证
- **办理时间**：周一至周五 8:30-11:30, 14:30-17:30
- **办结时间**：1个工作日

## 二、宿舍调整

### 2.1 申请条件
- 因疾病需要特殊照顾
- 宿舍成员关系不和
- 其他特殊情况

### 2.2 办理流程
1. 填写《宿舍调整申请表》
2. 所在学院签署意见
3. 提交学生处宿舍管理中心
4. 等待审批结果

### 2.3 办理地点
学生处宿舍管理中心（学生活动中心101室）

## 三、差旅报销

### 3.1 政策依据
- 《郑州航空工业管理学院差旅费管理办法》
- 公务出差需提前审批

### 3.2 报销流程
1. 登录财务系统填写报销单
2. 上传相关凭证（机票、车票、住宿发票等）
3. 部门负责人审批
4. 财务处审核报销

### 3.3 办理地点
财务处报销大厅（行政楼1楼）

### 3.4 联系人
财务处李老师：0371-61912345

## 四、户籍办理

### 4.1 户籍迁入
- **所需材料**：录取通知书、户口迁移证、身份证
- **办理地点**：保卫处户籍科（行政楼108室）
- **办理时间**：新生报到期间集中办理

### 4.2 户籍迁出
- **所需材料**：毕业证、就业报到证、身份证
- **办理地点**：保卫处户籍科
- **办理时间**：毕业季集中办理
