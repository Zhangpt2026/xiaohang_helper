# 应急防骗指南

## 一、防诈骗提示

### 1.1 常见诈骗手段
- 冒充老师/辅导员要求转账
- 冒充学校财务要求交学费
- 刷单返利诈骗
- 校园贷诈骗
- 中奖诈骗

### 1.2 防骗原则
- 凡是要求转账的都是诈骗
- 凡是要求提供验证码的都是诈骗
- 凡是中奖需要先交钱的都是诈骗
- 凡是自称公检法要求转账的都是诈骗

### 1.3 注意事项
- 不轻易透露个人信息（身份证号、银行卡号、密码等）
- 不轻易点击陌生链接
- 不轻易下载陌生APP
- 遇到可疑情况及时联系辅导员核实

### 1.4 举报方式
- 拨打110报警
- 联系学校保卫处：0371-61911402
- 通过"国家反诈中心"APP举报

## 二、心理援助

### 2.1 心理援助热线
- **全国心理援助热线**：12320-5
- **工作时间**：24小时

### 2.2 学校心理咨询中心
- **电话**：0371-61911203
- **办公地点**：学生活动中心301室
- **工作时间**：周一至周五 9:00-11:30, 14:30-17:30
- **服务内容**：心理咨询、心理测评、心理健康教育

### 2.3 紧急处理流程
1. 立即拨打心理援助热线12320-5
2. 联系学校心理咨询中心
3. 告知辅导员或班主任
4. 必要时联系家长

## 三、紧急事件处理

### 3.1 火警
- **报警电话**：119
- **校内电话**：0371-61911403（消防科）
- **处理流程**：立即撤离、拨打报警电话、不要乘坐电梯

### 3.2 医疗急救
- **报警电话**：120
- **校医院电话**：0371-61911600
- **处理流程**：评估伤情、拨打急救电话、必要时进行急救

### 3.3 治安事件
- **报警电话**：110
- **校内电话**：0371-61911402（治安科）
- **处理流程**：保护现场、拨打报警电话、配合调查

## 四、校园安全

### 4.1 安全须知
- 夜间出行要结伴而行
- 妥善保管个人财物
- 宿舍要及时锁门
- 注意食品安全

### 4.2 安全检查
- 定期检查宿舍安全隐患
- 发现可疑人员及时报告
- 配合学校安全检查
