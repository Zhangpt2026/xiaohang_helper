# 文件行数统计功能测试用例表
| 编号 | 测试操作 | 预期结果 | 实际结果 | 是否通过 |
| ---- | -------- | -------- | -------- | -------- |
| 1 | campus_data 中有 3 个 txt 文件 | 输出 3 个文件的行数 | 程序遍历目录，识别全部txt文件，分别打印每个文件名称与对应行数，统计总数正确 | 是 |
| 2 | 新增一个 dormitory.txt | 输出中增加 dormitory.txt | 刷新读取目录后，输出列表新增 dormitory.txt 及其文本行数 | 是 |
| 3 | 新增一个 readme.md | 不统计 md 文件 | 程序仅筛选后缀为.txt的文件，readme.md未参与遍历与行数统计 | 是 |
| 4 | 把文件夹名改错 | 提示文件夹不存在 | 代码捕获目录读取异常，控制台输出友好提示：目标文件夹不存在，请检查路径名称 | 是 |
